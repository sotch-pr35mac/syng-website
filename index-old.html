Syng | 词应

A free, open-source, cross-platform, Chinese-to-English dictionary

Features

Search

Search in English, Chinese (traditional and simplified characters), and even in pinyin. Inspect the component characters that make-up a word and even save searches to vocab lists.

Bookmarks

Save words to bookmarks and custom vocab lists, share vocab lists with classmates, and take notes on individual words.

Study

Study your vocab lists with Syng's automatically generated flashcards. Make sure you are learning on track with Syng's quiz feature.

Cross Platform

No matter your operating system, Syng will run on your computer. Syng works on macOS, Windows, and Linux, all without needing to connect to the internet to use.

Free

Syng is free to use and free to modify, now and forever. If you want to donate, you can here. You can also contribute to the project here.

Tools

A set of handy tools to turn characters into pinyin, turn tone numbers into tone marks, and to convert between simplified and traditional characters.

About

About Syng

Syng was created to be a simple, yet full-featured Chinese language study tool. Syng is free, cross platform, and open source, so that no matter how much money you have, or what computer operating system you use, you can enjoy Syng and all of its features. You won't find any pay to use premium features in Syng. In fact, if you think Syng should have a specific feature, you can go ahead and contribute to the project here.

About the Name

Syng is a stylized spelling of cíyīng, the pinyin for 词应. 词应 is short for 词典应用, literally translating to Dictionary App.

Developers Welcome

If you're interested in contributing to Syng in whatever capacity you can, feel free to contact the maintainer.

Using the Installer Script

To install Syng from the command line with the distro-agnostic installer, enter the following lines of code into your terminal. It's good to be cautious about what you are putting in your terminal, you can read the 64-bit install script here and the 32-bit install script here.

Linux 64-bit
wget -O- https://bit.ly/2IeuwBb | bash
Linux 32-bit
wget -O- https://bit.ly/2KiED8x | bash
